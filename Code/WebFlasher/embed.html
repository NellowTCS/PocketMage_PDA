Installation Options

If "Preserve user data" is checked, only the application partition will be flashed, keeping settings and saved content.

If unchecked, a full installation will erase all data and settings.

Requirements

Chromium-based browser with Web Serial API support (Chrome, Edge, Opera).

Connect device via USB and make sure no other program is using the serial port.
